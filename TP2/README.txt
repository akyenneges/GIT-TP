## Approfondir vos connaissances en Git

1. Installer Git sur votre ordinateur en suivant les instructions disponibles sur le site officiel de Git (https://git-scm.com/downloads).

2. Créer un nouveau dépôt Git en utilisant la commande git init dans le répertoire de votre choix.

3. Créer un fichier nommé "README.txt" dans le répertoire du dépôt, et y écrire un texte de votre choix.

4. Ajouter le fichier "README.txt" à l'index Git en utilisant la commande git add "README.txt".

5. Créer une nouvelle version du dépôt en utilisant la commande git commit -m "Initial commit".

6. Créer un nouveau fichier nommé "script.py" dans le répertoire du dépôt, et y écrire un script Python qui affiche un message à l'écran.

7. Ajouter le fichier "script.py" à l'index Git en utilisant la commande git add "script.py".

8. Créer une nouvelle version du dépôt en utilisant la commande git commit -m "Ajout du script Python".

9. Créer une nouvelle branche nommée "feature" en utilisant la commande git branch "feature".

10. Passer sur la branche "feature" en utilisant la commande git checkout "feature".

11. Modifier le fichier "script.py" en ajoutant une fonction qui calcule la factorielle d'un nombre, puis ajouter les modifications à l'index Git en utilisant la commande git add "script.py".

12. Créer une nouvelle version du dépôt en utilisant la commande git commit -m "Ajout de la fonction factorielle dans la branche 'feature'".

13. Revenir sur la branche principale en utilisant la commande git checkout master.

14. Fusionner la branche "feature" dans la branche principale en utilisant la commande git merge "feature".

15. Vérifier les différences entre les versions du fichier "script.py" en utilisant la commande git diff "script.py".

16. Revenir à une version précédente du fichier "script.py" en utilisant la commande git checkout "nom_de_la_version" "script.py".

17. Ajouter un nouveau fichier nommé "data.csv" dans le répertoire du dépôt, et y écrire des données au format CSV.

18. Ajouter le fichier "data.csv" à l'index Git en utilisant la commande `git add "data.

19. Créer une nouvelle version du dépôt en utilisant la commande git commit -m "Ajout du fichier de données".

20. Créer un nouveau fichier nommé "script2.py" dans le répertoire du dépôt, et y écrire un script Python qui lit les données contenues dans le fichier "data.csv" et affiche des statistiques sur ces données.

21. Ajouter le fichier "script2.py" à l'index Git en utilisant la commande git add "script2.py".

22. Créer une nouvelle version du dépôt en utilisant la commande git commit -m "Ajout du script d'analyse de données".

23. Pusher les modifications du dépôt sur un serveur distant en utilisant la commande git push "nom_du_serveur" "nom_de_la_branche".
